# 厂务管理系统 MVP v4.0

可直接部署到 GitHub Pages 的可操作前端 MVP。

## 运行
双击 `index.html`，或上传到 GitHub 仓库根目录并开启 Pages。

## 功能
- 单一图窗切换：区域三维总览 / 楼栋三维 / 楼层三维 / 楼层平面图
- 民企区域三栋楼总览
- 墙体横平竖直绘制，支持墙体宽度
- 层高设置，楼层三维图显示层高
- 配电箱和设备不同图标
- 配电箱进线、开关、上级配电箱、出线开关模板
- 设备类别和自定义字段
- 数据保存到浏览器 LocalStorage
- JSON 导入导出

## 注意
GitHub Pages 只能运行前端，数据保存在当前浏览器。正式使用需升级为后端数据库版本。
