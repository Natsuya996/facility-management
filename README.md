# 厂务管理系统 MVP Latest

## 运行方式
直接双击 `index.html`，或上传到 GitHub Pages。  
管理员账号：admin  密码：admin123

## 包含功能
- 管理员登录
- 区域 / 楼栋 / 楼层管理
- 选择楼层后查看俯视图、45°三维斜视图、楼栋三维图
- 楼层编辑模式：绘制墙体、上传平面图、自动生成示意墙体、填写层高并生成单层三维图
- 配电箱详细信息模板：进线电缆、总开关、上级配电箱、出线开关/回路
- 其他厂务设备：类别可编辑、自定义字段、保养计划
- 楼栋三维：根据俯视图尺寸和楼层层高生成，位置可编辑
- 本地浏览器保存、JSON导入导出
